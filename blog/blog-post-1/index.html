Installation and language configuration.

App installation

  • September 24, 2018
  • Android app , Acoustic variometer

Since the communication of the application with the pilot is carried out through voice messages, it is necessary to have the voice synthesis engine of your device correctly configured by installing the voice synthesis files corresponding to English, Spanish or Portuguese, which are the languages ​​supported by the application. If your device’s regional settings match any of those languages, that will be the default language the app will use. Otherwise it will use English as the default language. In the next step, the application looks for a voice synthesis file corresponding to the default language. If it cannot be found, it will display an error message telling you that you must install it and restart the application.

A voice synthesis file is a file that is used so that the speech synthesis engine can generate audible vocal messages from text. Each voice synthesis file will be associated with a language and dialect.

If you want to change the default language to one of the other two languages ​​supported by the application, you can modify it in the Language option on the settings screen. The application will then search for a voice synthesis file corresponding to the language you have selected. If it cannot be found, it will display an error message telling you that you must install it and restart the application.

On the main screen, under the heading Speaker language, the language of the voice messages is displayed or `Language files are missing’ if there is no voice synthesis file installed on the device.
