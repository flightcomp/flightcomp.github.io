How to set the application and voice messages language

App install prerrequisites

  • September 24, 2018
  • Android app , Acoustic variometer

The philosophy behind this app is to replace the visual interaction with your device by audible interaction made of sounds and voice messages. Therefore you won’t need to take your phone in sight during the flight or keep the screen on, which brings freedom (no cockpit) and battery life (screen on drains lots of battery power) to your flights. There are wired and wireless helmet headset kits that will allow you to interact with the app, answer phone calls during the flight and even connect a handheld transceiver.

Therefore, in order to use the application it is necessary to have correctly configured the speech synthesis engine of your device by installing the speech synthesis files corresponding to English, Spanish or Portuguese, which are the languages supported by the application. If the regional configuration of your device matches any of these languages the application is automatically configured for that language during the installation, otherwise it will use English. Then the application searches for a speech synthesis file corresponding to the application configuration language. If it does not find it, it stops and displays an error message telling the user to install it and restart the application. If the regional configuration of your device does not match any of the languages supported by the application and you do not want to use English, you can change the language in the application preferences to Spanish or Portuguese. The application will then search for a speech synthesis file corresponding to the language you have selected and if it does not find it, it stops and displays an error message telling the user to install it and restart the application.
