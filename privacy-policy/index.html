Information Collection and Use

Our website does not request or collect personal information of any kind. Our app collects your location data and stores it in a file with .igc extension in the io.github.flightcomp/trks folder of your device. When you close the application or put it in standby mode this file is closed and the app stops tracking your location. The application won’t access this file again. You can prevent the app from tracking your flight by just turning off location in your device google preferences or denying the access from the app. See next section. In any case the app will track your location while not in use.

App permissions

Since Android Marshmallow some actions need explicit permission from the user. With the intent to offer the best service, our app asks the user for these permissions. Any of these permissions can be revoked by the user at any time with the consequent loss of the associated functionalities.

  • Permission to know the exact location: This permission is required in order to provide high value-added functionalities as:

    • Estimation of wind speed and direction

    • Waypoint navigation.

    • Flight track recording.

    • Best glide, speed control.

    • Final glide assistant.

  • Permission to manage bluetooth connections in case you want to connect to an external module like Bluefly.

  • Permission to create a foreground service. The app can run with the screen off and is less prone to be closed by the Android OS in case of resource depletion.

  • Permission to access storage. Waypoint files, flight tracks and flight tasks are stored in the folder io.github.flightcom of your device so they can be read/wrote by other apps. For example: Use an IGC file reader app to view your flights. Without this permission, the navigation function will be disabled and the flights won’t be recorded.

  • Permission to bypass battery optimizations. Granting this permission to the app you avoid the phone entering in doze mode while the app is running.
