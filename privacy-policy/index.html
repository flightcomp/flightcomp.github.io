App permissions

With the intent to offer the best service, our app asks the user for this permissions. Any of these permissions can be revoked by the user at any time with the consequent loss of the associated functionalities.

  • Permission to know the exact location: This permission is required in order to provide high value-added functionalities as:

    • Estimation of wind speed and direction

    • Waypoint navigation.

    • Flight track recording.

    • Best glide speed control.

    • Final glide assistant.

  • Permission to manage bluetooth connections in case you want to connect to an external module like Bluefly.

  • Permission to create a background service. The app can run with the screen off and is less prone to fail.

  • Permission to access storage. Waypoint files, flight tracks and flight tasks are stored in the folder io.github.flightcom of your device so they can be read and stored by oter apps if you want. Read your flight records with the app IGCBrowser, for example. Withount it the navigation function are disabled and the flight won’t be recorded.
